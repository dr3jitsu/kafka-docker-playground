# FTPS Source connector



## Objective

Quickly test [FTPS Source](https://docs.confluent.io/current/connect/kafka-connect-ftps/source/index.html#ftps-source-connector-for-cp) connector.



## How to run


Simply run:

```bash
$ ./ftps-source-json.sh
```
